Detailed Description

Image controller.

Factory class for images.
Stores and controls all loaded images (read, remove, change etc.).
Manages the connection between main program logic and loaded image files.

Author
Ivan Menshikov (mensh.nosp@m.iva@.nosp@m.fit.c.nosp@m.vut..nosp@m.cz).

◆ readImage()

bool ImageFactory::readImage ( )

Reads image file from it's path, converts it to ASCII art and adds it to images.
If image file was opened and converted to ASCII art successfully, function will ask user to load another one.
Otherwise will ask user to try again.

Note
Function checks file existence and image format by itself.
Warning
Works only in console mode. If you are using UI, you should call UI::hide() first.
Returns
true if at least 1 image file was loaded and converted successfully, false otherwise.
