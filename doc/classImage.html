Detailed Description

Abstract base class for all supported image formats.

Holds all image data without compression.
Operates with image in common way, such as: convert to ASCII, add effect etc.

Author
Ivan Menshikov (mensh.nosp@m.iva@.nosp@m.fit.c.nosp@m.vut..nosp@m.cz).

◆ convertToAscii()

void Image::convertToAscii ( const std::string &  grayscaleLevel)

Converts all RGB pixels from rawData to Gray pixels with rgbToGray() function.
Finds needful ASCII characters in grayscaleLevel for all Gray pixels by their brightness.

Note
If isContrasted is set, then calls applyContrast() function.
If isNegatived is set, then calls applyNegative() function.
If isSharpened is set, then calls applySharpen() function.
Parameters
grayscaleLevelRange of symbols from darkest to lightest (more at ImageFactory::grayscaleLevel).
