Detailed Description

Provides panel View in user interface.

This class is derived from View and holds all data, needed for panel.
Operates only with panel.
Other operations are implemented in View class.

Author
Ivan Menshikov (mensh.nosp@m.iva@.nosp@m.fit.c.nosp@m.vut..nosp@m.cz).
