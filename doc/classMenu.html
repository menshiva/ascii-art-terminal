Detailed Description

Provides menu View in user interface.

This class is derived from View and holds all data, needed for menu.
Operates only with menu.
Other operations are implemented in View class.

Author
Ivan Menshikov (mensh.nosp@m.iva@.nosp@m.fit.c.nosp@m.vut..nosp@m.cz).
