Detailed Description

Abstract base class for all supported user interface elements.

Holds all data, needed for basic user interface element.
Operates with element in common way, such as: draw, resize, getHeight, getWidth, etc.

Author
Ivan Menshikov (mensh.nosp@m.iva@.nosp@m.fit.c.nosp@m.vut..nosp@m.cz).
