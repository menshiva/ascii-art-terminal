Detailed Description

User interface.

User interface logic.
Stores and controls all views (draw, resize, show etc.).
Manages connection between main program logic and views.
It is derived from View due to providing the same basic logic as the other views.

Author
Ivan Menshikov (mensh.nosp@m.iva@.nosp@m.fit.c.nosp@m.vut..nosp@m.cz).
