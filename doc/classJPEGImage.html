Detailed Description

JPEG (JPG) image format.

This class is derived from Image and holds all JPEG format data.
Operates only with reading data from image file.
Other operations are implemented in Image class.

Author
Ivan Menshikov (mensh.nosp@m.iva@.nosp@m.fit.c.nosp@m.vut..nosp@m.cz).
